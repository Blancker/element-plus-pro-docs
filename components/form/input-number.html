InputNumber 数字输入框 ​

数字输入框，基础 el-input-number 封装，增加了 "允许清空、幽灵模式" 等扩展功能。

基本用法 ​

点击查看代码
vue
<template>
  <s-input-number v-model="numberValue" placeholder="请输入"></s-input-number>
</template>

<script setup lang="ts">
  import { ref } from "vue";

  const numberValue = ref<number>(1);
</script>
